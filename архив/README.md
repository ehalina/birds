# Motion Recorder

Автоматическая запись видео при обнаружении движения.

## Веб-версия (рекомендуется)

Откройте `index.html` в браузере на Android устройстве.

### Функции
- Автоматическое обнаружение движения в кадре
- Автоматический старт записи при обнаружении движения
- Автоматическая остановка записи через 30 секунд без движения
- Сохранение видео на устройство

### Использование
1. Откройте `index.html` в браузере (Chrome, Firefox)
2. Разрешите доступ к камере и микрофону
3. Нажмите "Начать"
4. При обнаружении движения начнется автоматическая запись
5. Видео сохраняется автоматически при остановке записи

### Требования
- Браузер с поддержкой MediaRecorder API (Chrome, Firefox на Android)
- HTTPS или localhost (для доступа к камере)

## Android приложение

### Функции
- Автоматическое обнаружение движения в кадре
- Автоматический старт записи при обнаружении движения
- Автоматическая остановка записи через 30 секунд без движения
- Сохранение видео в галерею

### Требования
- Android 7.0 (API 24) или выше
- Разрешения: камера, микрофон

### Сборка
```bash
./gradlew assembleDebug
```

### Установка
Установите APK на устройство через ADB или Android Studio.
